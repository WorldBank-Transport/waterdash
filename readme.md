Water Dashboards
================

[![Build Status](https://travis-ci.org/WorldBank-Transport/waterdash.svg?branch=master)](https://travis-ci.org/WorldBank-Transport/waterdash)


## Requirements

* IOJS >= 3 or NodeJS >= 4


## Installation

```bash
$ npm install
```

## Build

```bash
$ npm run build
```

## Development

### Run the dev server

```bash
$ npm start
```

Will serve the app at [`localhost:3000`](http://localhost:3000).

### Lint and test

```bash
$ npm test
```

#### Lint only

```bash
$ npm run lint
```

#### Test only

```bash
$ npm run test-only
```
